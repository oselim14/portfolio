Projects

Mancala

Mancala is an HTML 5 browser based game, with CSS for presentation and JavaScript for game logic. Mancala allows the user to play this classic game against a randomly generated computer or a friend from the same device. It has responsive design, and will fit any screen size it is displayed on.

Pocket Monsters

Pocket Monsters is a full-stack application using NodeJS, Express and MongoDB, as well as a third party API to create a database that allows users to search for Pokemon information and store the information in their individual account.

Films&Flix

Films&Flix is a website that allows users to search for movies, and review the movies. The movie search pulls information about the movie from the IMDb API and allows the user to see title, directors, writers, cast, etc. Users are then able to leave a review on the movie, and see other reviews for the movie as well. As a visitor, they can also search movies and see details, visitors are only not able to create reviews.
